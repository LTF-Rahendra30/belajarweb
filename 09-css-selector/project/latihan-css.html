Apa itu Blockchain?

Blockchain adalah struktur data terdesentralisasi yang menyimpan transaksi dalam blok-blok yang saling terhubung secara kriptografis.

  • Desentralisasi
  • Immutable (tidak bisa diubah)
  • Transparan

CSS Flexbox vs Grid

Flexbox dan Grid adalah dua sistem layout CSS modern. Masing-masing punya kekuatan di use-case yang berbeda.

  • Flexbox: layout 1 dimensi
  • Grid: layout 2 dimensi
  • Bisa dikombinasikan
  • Support browser modern
